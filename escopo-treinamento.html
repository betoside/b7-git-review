#8: Diferença entre arquivos


Exibir os detalhes diferentes entre os arquivos:
betosideArena:b7-git-review betoside$ git diff

Exibir apenas o nome dos arquivos que foram alterados:
betosideArena:b7-git-review betoside$ git diff --name-only

Descartar as alterações do arquivo:
betosideArena:b7-git-review betoside$ git checkout HEAD -- escopo.txt
